\# Uso de IA generativa



Durante el desarrollo de BarberBot se utilizó IA generativa como apoyo para:



\- Estructurar la arquitectura del proyecto.

\- Diseñar prompts para interpretar mensajes de clientes.

\- Generar ejemplos de pruebas.

\- Apoyar la documentación técnica.

\- Revisar fragmentos de código y lógica del sistema.



El equipo revisó, adaptó y comprendió los elementos generados antes de incluirlos en el proyecto.
