# Condiciones del servicio de Fenix BOT

Última actualización: 29 de junio de 2026

Estas Condiciones del servicio regulan el uso de Fenix BOT, un bot diseñado para atender mensajes y gestionar solicitudes a través de WhatsApp.

Al interactuar con Fenix BOT, el usuario acepta estas condiciones.

## 1. Descripción del servicio

Fenix BOT permite a los usuarios enviar mensajes por WhatsApp para realizar consultas, solicitar información, gestionar citas, hacer reservas, modificar horarios o cancelar solicitudes, según la disponibilidad del servicio.

## 2. Uso permitido

El usuario se compromete a usar Fenix BOT únicamente para fines legítimos y relacionados con la atención, consulta o gestión de servicios ofrecidos por JS Brots Company.

El usuario no debe usar el bot para:

- Enviar mensajes ofensivos, ilegales, abusivos o fraudulentos.
- Intentar acceder sin autorización a sistemas, cuentas o datos.
- Enviar spam o mensajes automatizados no autorizados.
- Suplantar la identidad de otra persona.

## 3. Disponibilidad del servicio

Fenix BOT puede estar disponible de forma continua, pero no se garantiza que el servicio funcione sin interrupciones, errores o retrasos.

El servicio puede suspenderse temporalmente por mantenimiento, fallos técnicos, cambios en plataformas externas o causas fuera de nuestro control.

## 4. Exactitud de la información

Fenix BOT busca responder de forma clara y útil. Sin embargo, algunas respuestas pueden depender de la información proporcionada por el usuario o de la disponibilidad real del servicio.

Las reservas, citas o cambios solo se consideran confirmados cuando el sistema o el responsable del servicio lo indique expresamente.

## 5. Responsabilidad del usuario

El usuario es responsable de proporcionar información correcta, especialmente datos como fecha, hora, servicio solicitado y número de contacto.

JS Brots Company no se hace responsable por errores derivados de información incorrecta, incompleta o confusa enviada por el usuario.

## 6. Tratamiento de datos

El tratamiento de datos personales se realiza conforme a la Política de privacidad de Fenix BOT, disponible en la página correspondiente.

## 7. Modificaciones del servicio

Podemos modificar, suspender o retirar funcionalidades de Fenix BOT en cualquier momento, según necesidades técnicas, comerciales o de seguridad.

## 8. Terminación del uso

Podemos limitar o bloquear el uso de Fenix BOT si detectamos abuso, uso indebido, actividades fraudulentas o incumplimiento de estas condiciones.
